How use the gswebPlayer

Quick Description

gswebPlayer is a player in which you can preview the video without having to create covers or animated gifs.

Before starting

Download the last version of the gswebPlayer.

Download

To start using gswebPlayer it is necessary to declare an element and assign it an ID. 

<div id="video"></div>
gswebPlayer only detects elements with an ID.

after this, you need to add the following script before closing the body

<script src='https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js'></script>
<script src="https://juanjimeneztj.github.io/cdn/js/gswebPlayer/gswebPlayer.min.js" ></script>
Jquery is necessary to run this module.

Finally, if the source/video is in ROGUE's aws, it is not necessary to add the domain since gswebPlayer will search for it automatically, you must create the object and assign it the necessary parameters as shown below

<script>
  $('#video').gswebPlayer({
    src: 'SigmaSurges/Tom+Endorse+Sigma+Surges.mp4'
  });
</script>

Otherwise, if the source of the video is external (MP4 obligatorily) it would be configured as follows:

<script>
  $('#video').gswebPlayer({
    domain: 'https://juanjimeneztj.com/',
    src: 'media/Placeholder_Video_1080p.mp4'
  });
</script>

Parameter Description
src This parameter receives the source of the video, if it is a video hosted by ROGUE on AWS, the player automatically recognizes the source, so it is not necessary to add the "domain" parameter.

Important note: This player only accepts MP4 type video source, if you try to add any other type of file it is likely that the player will not work properly.
domain This parameter is used to add the domain of an external source to ROGUE, for example, if you want to add the following url, it would be as follows: 
<script>
  $('#video').gswebPlayer({
    domain: 'https://juanjimeneztj.com/',
    src: 'media/Placeholder_Video_1080p.mp4'
  });
</script>
progress This parameter is used to show or hide the progress bar of the video, by default it is false so that it is not shown, but if you want it to be shown you can add the parameter with the value of true.
buttonText This parameter is used to add the text to the button that will appear below the video.
buttonSrc This parameter is used to add the url to the button that will appear below the video.
buttonClass This parameter is used to add classes to the button that will appear below the video.
buttonTime This parameter is used to configure the time at which the button will appear below the video.

Important note: This parameter must be entered in seconds only.
window This parameter is used to configure if the button below the video will open in a new tab, by default the value is false so that the link is opened within the same window, but it can be added and changed to true so that the link is opened in a new tab without replacing the current one.
If any of the following two parameters is not added to the player parameters, the button will not be shown.

  • buttonText
  • buttonSrc
