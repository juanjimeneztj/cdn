How use the gswebPlayer

Quick Description

gswebPlayer is a player in which you can preview the video without having to create covers or animated gifs.

Before starting

Download the last version of the gswebPlayer.

Download

To start using gswebPlayer it is necessary to declare an element and assign it an ID.

<div id="video"></div>
gswebPlayer only detects elements with an ID.

after this, you need to add the following script before closing the body

<script src='https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js'></script>
<script src="https://juanjimeneztj.github.io/cdn/js/gswebPlayer/gswebPlayer.min.js" ></script>
Jquery is necessary to run this module.

Finally, if the source/video is in ROGUE's aws, it is not necessary to add the domain since gswebPlayer will search for it automatically, you must create the object and assign it the necessary parameters as shown below

<script>
  $('#video').gswebPlayer({
    src: 'SigmaSurges/Tom+Endorse+Sigma+Surges.mp4'
  });
</script>

Otherwise, if the source of the video is external (MP4 obligatorily) it would be configured as follows:

<script>
  $('#video').gswebPlayer({
    domain: 'https://juanjimeneztj.com/',
    src: 'media/Placeholder_Video_1080p.mp4'
  });
</script>
